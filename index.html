🔌 Connect Rojo to Squeeze
Rojo syncs local files into Roblox Studio in real-time. Squeeze writes directly to your Rojo project folder — when rojo serve is running, changes appear in Studio instantly.
1
Install Rojo CLI
Install via Aftman (recommended) or Foreman.
aftman add rojo-rbx/rojo@7 aftman install
2
Install Rojo Plugin in Studio
Open Roblox Studio → Plugin Manager → search "Rojo" → Install. Or get it from the Creator Store.
3
Init your project
Create a folder and initialize Rojo inside it.
mkdir my-game && cd my-game rojo init
4
Start Rojo server
Run rojo serve in your project folder. Then in Studio, click "Connect" in the Rojo plugin.
rojo serve # Studio plugin → Connect → localhost:34872
5
Connect folder in Squeeze
Click "Connect Folder" in the Explorer panel and select your Rojo project folder. Squeeze will read/write files there — Rojo picks up changes automatically.
How it works
Squeeze ↔ Your local files ↔ Rojo CLI (watches for changes) ↔ Studio Plugin (syncs into game). When the AI generates a script, Squeeze saves it to src/server/, src/client/, or src/shared/ — and Rojo syncs it to Studio within seconds.
