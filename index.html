🔌 Rojo Integration
Rojo is a Roblox Studio plugin that syncs your local files into your game in real-time. Squeeze generates Rojo-compatible scripts you can drop straight into your project.
1
Install Rojo Plugin in Studio
Open Roblox Studio → Plugin Manager → search "Rojo" and install it. Or install from the Roblox Marketplace.
2
Install Rojo CLI on your machine
Install via Aftman (recommended) or download from the Rojo GitHub releases page.
aftman add rojo-rbx/rojo aftman install # OR for Windows: scoop install rojo
3
Initialize your Rojo project
Create a new folder for your game, then run:
mkdir my-roblox-game cd my-roblox-game rojo init
4
Place Squeeze-generated scripts
Scripts generated by Squeeze map to the correct Rojo directories. ServerScripts go to src/server/, LocalScripts to src/client/, and ModuleScripts to src/shared/.
5
Start the Rojo server & connect
Run the server, then click Connect in the Studio plugin. Changes sync instantly.
rojo serve # Studio plugin: click "Connect" → localhost:34872
